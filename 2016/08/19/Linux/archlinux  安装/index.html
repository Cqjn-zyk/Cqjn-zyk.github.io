ArchLinux 安装流程

  1. 1. A. 安装前的准备
  2. 2. B.开始安装
  3. 3. C.配置新系统
  4. 4. D.安装配置常用软件环境
  • 官方Wiki页面
  • Arch中文论坛–安装指南

A. 安装前的准备

  • 1.键盘布局

    默认键盘布局是 US 键盘, 可以通过 loadkeys “keymap_name” 的命令切换键盘布局，键盘映射表请查看 /usr/share/kbd/keymaps
    一般不需要改动

    localectl list-keymaps  //查看可用的键盘列表
loadkeys <key_map>
  • 2.连接网络
    • 有线网络都已经启用了dhcpcd服务，一般不需要设置即可联网
    • 无线网络运行 wifi-menu 设置无线网络
      wifi-menu
    • 设置完成以后运行 ping www.baidu.com 查看网络是否畅通
  • 3.硬盘分区
    fdisk  /dev/sda
  • 4.格式化分区
    mkfs.ext4 /dev/sda*  
mkswap //swap 分区
  • 5.挂载分区
    mount /dev/sda* /mnt  
mount /dev/sda* /mnt/home  //if have
mount /dev/sda* /mnt/boot 
swapon /dev/<swap 分区>

B.开始安装

  • 6.选择镜像
    vim /etc/pacman.d/mirrorlist
//添加国内比较快的镜像源，一般使用网易的镜像源
Server = http://mirrors.163.com/archlinux/$repo/os/$arch
//USTC的源
Server = http://mirrors4.ustc.edu.cn/archlinux/$repo/os/$arch
  • 7.同步数据库
    pacman -Syy 
同步远程软件包列表到本地
  • 8.安装基本系统
    pacstrap /mnt base base-devel  //  /mnt 目录就是安装好系统的根目录
  • 9.配置fstab
    genfstab -U -p /mnt >> /mnt/etc/fstab  //生成分区信息并写入到新系统

C.配置新系统

  • 10.切换到新系统
    arch-chroot /mnt /bin/bash
  • 11.设置主机名
    echo computer_name /etc/hostname
vi /etc/hosts    //将主机名添加到最后
  • 12.设置语言
    vi /etc/locale.gen  //反注释掉所需要的locale
locale-gen       //生成locale
echo LANG=<locale> > /etc/locale.conf //设置默认locale
  • 13.设置时区
    ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
  • 14.设置网络
    • 有线网络
      systemctl start dhcpcd 
systemctl enable dhcpcd  //开机启动dhcp服务
    • 无线网络
      pacman -S iw wpa_supplicant dialog wifi-menu
wifi-menu
  • 15.设置密码
    passwd
  • 16.安装引导程序
    pacman -S grub os-prober
grub-install --recheck /dev/sd*
grub-mkconfig -o /boot/grub/grub.cfg
  • 17.卸载分区重启
    exit
umount -R /mnt 
reboot

D.安装配置常用软件环境

  • 18.添加新用户
    useradd -m -g users -s /bin/bash <用户名>
//创建一个用户， 指定shell 主用户组为users ，创建用户主目录
passwd UserName  //设置密码
  • 19.安装显卡驱动
    首先要知道自己的显卡型号，然后在安装驱动包
    lspci | grep VGA
    官方驱动包：
型号 驱动包 备注
通用 xf86-video-vesa NA
Intel xf86-vidio-intel NA
Nvdia xf86-video-nouveau NA
AMD/ATI xf86-vedio-ati NA
  • 20.安装X窗口系统
    pacman -S xorg-server xorg-server-utils 
pacman -S xf86-input-synaptics //触摸板支持
pacman -S ttf-dejavu wqy-microhei //字体
  • 21.安装桌面环境
    pacman -S 桌面环境
* cinnanon
* Enlightement
* gnome
* kde
* lxde
* xfce
  • 22.安装登录管理器
    pacman -S 登录管理器
* lxdm
* gdm
* slim
* sddm
。。。。。。
systemctl enable 登录管理器 //设置自启动

到这里差不多就可以重启进入图形界面了。^^_^^
