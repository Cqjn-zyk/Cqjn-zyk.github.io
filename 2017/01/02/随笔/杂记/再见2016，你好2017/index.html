再见2016，你好2017

  1. 1. 1~2月期末和寒假
  2. 2. 3~4月公司实习期
  3. 3. 5~6月最后的校园时光
  4. 4. 7~12月新入职场

不知不觉一年就过去了，在这一年里，有迷茫，有欢乐，有憧憬。。。。。。下面将这一年的大大小小的事情按照时间顺序简单罗列一下：

1~2月期末和寒假

现在回想起来，1月份的时候应该还是在学校的时候，应该是大四上学期期末的时候，那时候应该还在忙着准备期末考试呢，还依稀记得，当时老师说这是最后一节课了，感觉心里还是蛮失落的，毕竟从小到大上了这么多年的学，马上没有上课，点到了，期末考试之后，在学校剩下的日子就是吃吃喝喝，好好珍惜一下和同学之间的日子了，毕竟以后再想相见就不容易了。后面放寒假了，我也就回家了，在家里的日子总是特别短，虽然这个假期是我在家过得最长的寒假了，转眼间，就快到了去公司报道的日期了，那时候满怀憧憬和担心，憧憬的是第一次正式踏入职场，能够自己挣钱养活自己了，不用在花家里的钱了，还是蛮开心的，担心的是自己第一次去魔都，能不能自己照顾自己还是个未知呢！废话连篇，终于到了去公司的日子，告别了家人，独自一人坐上了去往上海的火车，开启了新的征程，fighting !

3~4月公司实习期

在家颓废了这么多天，坐了一夜的火车，终于到了魔都，下了火车，只觉得自己和这个城市格格不入，毕竟我只是个来自偏僻小县城的农村人，坐在地铁上，明显感受到大城市的包容和进步，哈哈，出了地铁站，坐上了去往公司指定酒店的公交车，大概5,6分钟之后就到了公司订的莫泰酒店，坐车坐了一夜，好累，出示了身份证报上公司的名字就给安排了一个二人间，我和一个我们学校的同学一起住，等人到齐了后，第二天，就正式去公司了，第一周主要是了解公司的文化和一些具体政策，第二周被下放到了部门里面。开始学习工作用到的知识了。接下来就是趁着周末去找房子了,公司帮忙订的房子只有两周的时间可以住，虽然之间出了些波折，最后终于算是在公司附近找到了个不错的房子，走路到公司只需2,30分钟的时间，接下来就是慢慢学习了，慢慢的也接触到了项目。。。。转眼间，就得到学校完成论文了，哎，烦人那，真不想写论文啊。。。

5~6月最后的校园时光

回学校还是很开心的，毕竟是最后的校园时光了，在学校的生活就是写写论文，吃吃饭，聊聊天，好好享受了一下这最后自由的时光。

7~12月新入职场

这几个月所做的事情我总结了下大概分为以下几件事：

  • 能够独立完成MTK 平台下camera驱动的添加
  • 能够解决简单的camera bug
  • 开通了自己的博客，在茫茫网海中有了一个自己的小家
  • 参加了一次部门里的聚餐和游玩，好开心
  • 买了本李笑来老师的和时间做朋友
  • 哦，还有件事，参加了锤子发布会，见到了老罗哈哈
  • 大概了解了MTK平台下camera Hal层和驱动之间的流程
  • 恩，应该没有啥事了吧，还给家里买了部手机
  • 剩下有事的话等想起来再说吧
